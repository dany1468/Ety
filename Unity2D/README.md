## 楽しく学ぶ Unity2D超入門講座を読んだ後に

本で作ったスクリプトはサポートサイトで入手できます。

https://book.mynavi.jp/supportsite/detail/9784839966768.html

このリポジトリでは、本で作ったスクリプトを改良したり、自主制作のゲームを作るための少し機能を追加したスクリプトを公開しています。
